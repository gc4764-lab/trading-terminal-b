# Stock Trading Application

A comprehensive stock trading platform with multi-broker support, real-time charts, risk management, and multi-monitor capabilities.

## Features

- 📊 **Real-time Charts** - Interactive candlestick charts with technical indicators
- 🔔 **Smart Alerts** - Price, volume, and percentage change alerts
- 📈 **Portfolio Management** - Track positions, holdings, and performance
- ⚠️ **Risk Management** - Advanced risk metrics and position sizing
- 🔌 **Multi-Broker Support** - Connect to Zerodha, Upstox, Angel One
- 🖥️ **Multi-Monitor** - Detachable windows for multi-display setups
- 🎨 **Customizable UI** - Themes, layouts, and runtime configuration
- 📰 **Live News** - Latest market news from free APIs
- 📱 **Cross-Platform** - Windows, macOS, Linux support
